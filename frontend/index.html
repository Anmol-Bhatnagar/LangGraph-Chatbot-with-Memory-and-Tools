System Architecture

How the dual-memory LangGraph agent works under the hood.

1

Load Memories

Fetches all stored long-term facts from SQLite and injects them into the system prompt so the LLM has persistent recall.

2

Chatbot Node

Invokes the selected LLM (Gemini or Groq) with the enriched system prompt + conversation history. Streams the response token-by-token.

3

Extract Memory

Analyzes the latest exchange and decides whether to extract new user facts or preferences to save into SQLite long-term memory.

4

Trim History

When message count exceeds the short-term limit, prunes older messages and summarizes their content into key facts before removing them.
